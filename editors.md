# Editing

## Viewers

    - **cat**
    - **less** 
      - allows scrolling
      - **q** to quit

## Editors

    - Vi
      - on most systems
      - archaic
      - cumbersome
    - Nano
      - based on **pico** from UW
      - not incredibly powerful
      - easy to use 
    - Emacs
      - early editor
      - written by the GNU project people
      - difficult to use
      - not installed by default on Linux systems
    - gedit
      - uses GUI
