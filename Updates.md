# Software updates

# Updating system
    - **yum**
      - deprecated
        - replaced by dnf
      - **sudo yum update yum**
      - **sudo yum update**
      - **sudo yum install**
    - **dnf**
      - **sudo dnf update**
    - **apt**
      - for Debain distros
      - **sudo apt-get update**

# insalled packages

# uninstalling
    - **dnf remove**
    - rpm -e
      - directly remove from redhat package manager
